## Cohort-Retention-Analysis

Когортный анализ удержания клиентов на основе транзакционных данных магазина H&M.

## Основная цель работы:

* сформировать клиентские когорты по месяцу первой покупки;
* рассчитать retention;
* выявить сезонные эффекты;
* провести сегментацию удержания по возрастным группам;
* сформулировать продуктовые выводы на основе полученных результатов.

## Используемый датасет

Источник: https://www.kaggle.com
Датасет: H&M Personalized Fashion Recommendations

## Используемые файлы:

* `transactions_train.csv`
* `customers.csv`

В данных содержатся:

* даты покупок;
* идентификаторы клиентов;
* история транзакций;
* возраст клиентов.

## Этапы анализа

### 01. Первичный анализ данных

На первом этапе выполнен EDA:

* анализ периода данных;
* подсчёт числа клиентов;
* исследование распределения покупок на клиента.

---

### 02. Формирование когорт

Когорты формируются по месяцу первой покупки клиента.

Для каждой транзакции рассчитываются:

* дата первой покупки;
* когорта (`год-месяц`);
* число дней с момента первой покупки.

---

### 03. Расчёт retention

Для каждой когорты рассчитано удержание:

* D7
* D30
* D90

Используется cumulative retention:

* клиент считается удержанным, если совершил хотя бы одну повторную покупку в пределах выбранного горизонта.

---

### 04. Визуализация retention

Построена heatmap-карта удержания по когортам.

Она позволяет увидеть:

* сезонные колебания;
* различия между когортами;
* изменение retention во времени.

---

### 05. Сегментация по возрасту

Дополнительно проведён анализ удержания по возрастным группам:

* до 25 лет;
* 25–35 лет;
* 35–50 лет;
* старше 50 лет.

Для проверки различий использован статистический критерий хи-квадрат.

---

### 06. Product Insights

На финальном этапе сформулированы продуктовые выводы:

* общий тренд удержания;
* сезонные эффекты;
* влияние возраста;
* возможные причины различий между когортами.

## Основные выводы

* для поздних когорт наблюдается снижение удержания;
* сезонность влияет на вероятность повторной покупки;
* возрастные различия статистически значимы;
* практическое влияние возраста на удержание умеренное;
* дата первой покупки влияет на удержание сильнее, чем возраст клиента.
